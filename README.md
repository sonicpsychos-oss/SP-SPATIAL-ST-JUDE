# SP-SPATIAL-ST-JUDE · Visual Prototype Build

This repository now includes an executable build that generates the three required visual mockup frames from the creative brief.

## Build

```bash
make build
```

## Outputs

- `dist/frame1_donor_reveal.svg` — donor reveal macro view with warm lighting, context massing, family commons, sky-bridge, and interior-mapping suggestion.
- `dist/frame2_trauma_informed_pivot.svg` — split-screen mobile exploration vs focus mode with high-contrast Pantone 108C wayfinding path.
- `dist/frame3_feasibility_triptych.svg` — beauty, topology/wireframe, and fabrication-ready clay/nozzle technical triptych.

## Notes

- Red is reserved for the logo marker only.
- Wayfinding path uses a yellow Pantone 108C approximation.
- Geometry language demonstrates modular repetition and context-aware simplification.
